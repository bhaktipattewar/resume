Project 3

Smart Traffic Management System

This project focuses on optimizing urban traffic flow using density-based signal control. By placing IR or ultrasonic sensors at intersections, the system detects vehicle density on each road and dynamically adjusts signal timings—giving longer green lights to roads with heavier traffic.Future enhancements include AI-based traffic prediction, emergency vehicle prioritization, and pollution monitoring, making it a scalable solution for smart city traffic management.

Project 4

AI-Powered Resume Screening System

This project uses Natural Language Processing (NLP) and Machine Learning algorithms to screen and rank resumes based on job descriptions. The system analyzes keywords, skills, experience, and educational qualifications using libraries like spaCy or NLTK. A classification model then matches resumes to job roles and gives each a relevance score. It helps recruiters automate the shortlisting process. Project 5

Real-Time Sign Language Detection System

This project uses computer vision and deep learning to recognize hand gestures in real time and translate them into readable text or speech. By using a webcam, the system detects sign language gestures through a trained CNN model and displays the corresponding meaning instantly. It’s built with Python, OpenCV, and TensorFlow, and helps improve communication for people with hearing or speech impairments. The system can be accessed via a simple interface and even expanded to support multiple languages or mobile use.
